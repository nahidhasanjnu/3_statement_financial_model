# 3-Statement Financial Model

This project is a fully integrated financial model built using Microsoft Excel.

## Overview
This model links all three financial statements:
- Income Statement
- Balance Sheet
- Cash Flow Statement

## Key Features
- Dynamic linking between statements
- Assumption-driven forecasting
- Automated calculations
- Structured and clean design

## Skills Demonstrated
- Financial Modeling
- Financial Statement Analysis
- Forecasting
- Excel

## File
- 3_statement_model.xlsx

## How to Use
1. Open the Excel file
2. Modify assumptions
3. Observe impact on financial statements
